# 🛒 ElectroCart

ElectroCart is a simple and modern e-commerce website built with HTML, CSS, JavaScript, and Firebase.  
It features:

- 🔐 Firebase Authentication
- 📦 Product listing & filtering
- 🖼️ Banner and product image sliders
- 🔍 Search bar, categories, and more

### 🔗 Live Website:
👉 [Click here to view ElectroCart](https://pawanbadak.github.io/Electrocart/)

---

### 🚀 Technologies Used
- HTML, CSS, JavaScript
- Firebase (Auth, Firestore, Storage)
- SwiperJS for banners
